# 🚀 Distributed Inventory Management Service - Guía de Ejecución

## 📦 Ejecución con Docker (Recomendado)

### Prerrequisitos
- Docker 20.10+
- Docker Compose 2.0+

### Ejecución rápida
#### Build
```bash
docker-compose up --build
```
#### Logs
```bash
docker-compose logs -f
```
#### Down
```bash
docker-compose down
```
